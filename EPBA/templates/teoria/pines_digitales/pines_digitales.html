Pines digitales


¿Qué es?

Como Arduino es un sistema embebido, está compuesto de bastantes elementos que en este caso permiten una interacción con el exterior, pues podemos controlar dispositivos que van a tomar acción sobre cosas que pasan a nivel físico y también podemos leer variables del medio ambiente o de otros sistemas que sirven para luego ser analizadas y hecho esto, tomar decisiones para realizar una acción sobre ello. Por esta razón, el Arduino tiene lo que se denominan 13 pines digitales los cuales van desde el 1 hasta el 13, y quienes permiten leer valores booleanos o binarios, ya que solo pueden leer si existe un voltaje (5v) o no existe voltaje (0v); o permiten tener salidas de 0 y 5 voltios para de esta manera controlar elementos exteriores como motores, luces, bocinas, entre otros. Dependiendo de si se tiene un voltaje o no el valor será HIGH o LOW y gracias a esto, se puede tener un comunicación en la que dependiendo de los pulsos tendrá diferentes mensajes contenidos en ellos para de esta manera tomar decisiones o acciones frente a dicha entrada. Como ya se mencionó, estos pines digitales pueden ser de entrada o de salida, por lo que se pueden configurar de diferentes formas para darle diferentes usos.








Sintaxis

Para controlar los pines digitales hay que tener varias cosas en cuenta. Primero se sabe que el pin se puede almacenar en una variable que tenga el nombre del parámetro que se va a leer o del dispositivo que se desea controlar y también se podría poner directamente el número de dicho pin para realizar cualuier acción. Segundo, para realizar la lectura de cualquiera de estos pines se debe utilizar el comando digitalRead(), el cual tiene como parámetro el nombre de la variable donde se almacenó el pin o el número del pin directamente como ya se había mencionado. Una recomendación nuevamente, es que la lectura de dicho pin se almacene en una nueva variable para posteriormente realizar diferentes operaciones o acciones con ella. Por último, para enviar una señal a ya sea de 0 voltios, 5 voltios o de PWM, se debe usar el comando digitalWrite(), el cual tiene como parámetros el nombre de la variable donde se almacenó el pin o el número del pin directamente como ya se había mencionado, seguido de una coma y el valor de voltaje que se le quiere dar, ya sea alto o bajo (HIGH o LOW); o un valor de PWM que puede tomar valores entre 0 y 255.

Para leer el pin:
            
nueva_variable = digitalRead(pin_digital);
Para enviar una señal al pin:
            
digitalWrite(pin_digital, valor);
