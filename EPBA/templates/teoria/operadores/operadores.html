Operadores


¿Qué son?

Los operadores son un tipo de símbolo o carácter que pueden aparecer en las expresiones u operaciones, estos indican al programa la realización de determinadas operaciones lógicas o matemáticas. Estos se pueden aplicar a variables u otros objetos denominados operandos los cuales luego de ser operados van a arrojar valores ya sea de tipo numérico o booleano, dependiendo cuál sea el caso.

Operadores aritméticos

Los operadores aritméticos son aquellos que se representan con símbolos y con los cuales, dependiendo de su posición, realizan una operación matemática u asignación a los diferentes datos que se deben de operar para la obtención de los resultados solicitados o los cuales dan solución a un problema.

Operadores lógicos

Los operadores lógicos son un tipo de símbolo que puede aparecer en las expresiones lógicas, e indican al compilador la realización de determinadas operaciones lógicas, de las cuales se van a obtener valores booleanos ya sea, true o false, 1 o 0 y, en el caso de Arduino high o low. Con ese resultado se puede dar control a un programa utilizando las demás estructuras que brindan dicha posibilidad.

Operadores relacionales

Los operadores relacionales son los que permiten asignar o dar un valor o valores específicos a una variable para de esta manera hacer saber al programa el valor de esta y posteriormente operar con ella o compararla con otras variables.

Sintaxis

Para cada operador puede cambiar un poco la sintaxis, por lo que se mostrarán los casos que se pueden presentar en un programa y de esta manera dejar claridad en el alumno de cómo se deben usar y en qué casos. Para empezar, se tienen los operadores aritméticos, estos se dividen en dos, los unidireccionales y los bidireccionales. Los unidireccionales son aquellos que tienen operando a su lado derecho y se usan para dar un sentido a los valores, a continuación se muestra cómo: 

            
int numero = -18;
int dato = +18;

Por otra parte, los bidireccionales son los que tienen operandos a cada lado y son estos los que permiten realizar operacionas matemáticas tales como la suma, resta, multiplicación y división, en este caso, en la variable se guarda el resultado de la operación que se realice, en el siguiente fragmento se muestra cómo: 

            
int numero = (20-18);

Ahora, en los operadores relacionales, solo basta con darle un valor a una variable, de esta manera la variable toma el valor que se le asigne tal y como se muestra a continuación: 

            
int numero = 20;

Por último, se tienen los operadores lógicos, estos son únicamente bidireccionales, es decir, tiene un operando a cada lado y de esta manera evalúa si una expresión es falsa o verdadera. En el siguiente fragmento se muestra cómo: 

            
bool estado = (20>=10);

Tablas de operadores


Nombre Operador abreviado Significado
Asignación x = y x = y
Asignación de adición x += y x = x + y
Asignación de resta x -= y x = x - y
Asignación de multiplicación x *= y x = x * y
Asignación de división x /= y x = x / y
Asignación de residuo x %= y x = x % y

Operador Descripción Ejemplo
Igual (==) Devuelve true si los operandos son iguales. 3 == var1
Diferente (!=) Devuelve true si los operandos son diferentes. var1 != 4
Mayor que (>) Devuelve true si el operando izquierdo es mayor que el operando derecho. var2 > var1
Mayor o igual que (>=) Devuelve true si el operando izquierdo es mayor o igual que el operando derecho. var2 >= var1
Menor que (<) Devuelve true si el operando izquierdo es menor que el operando derecho. var1 < var2
Menor o igual que (<=) Devuelve true si el operando izquierdo es menor o igual que el operando derecho. var1 <= var2

Operador Descripción Ejemplo
Residuo (%) Operador binario. Devuelve el resto entero de dividir los dos operandos. 12 % 5 devuelve 2
Negación unaria (-) Operador unario. Devuelve la negación de su operando. Si x es 3, entonces -x devuelve -3.
Positivo unario (+) Operador unario. Intenta convertir el operando en un número, si aún no lo es. +"3" devuelve 3. +true devuelve 1.

Operador Uso Descripción
AND Lógico (&&) expr1 && expr2 Cuando se usa con valores booleanos, && devuelve true si ambos operandos son true; de lo contrario, devuelve false.
OR lógico (||) expr1 || expr2 Cuando se usa con valores booleanos, || devuelve true si alguno de los operandos es true; si ambos son falsos, devuelve false
NOT lógico (!) !expr Niega la expresión o el valor al que se le antepone la negación.
