Pines análogos


¿Qué es?

Como Arduino es un sistema embebido, está compuesto de bastantes elementos que en este caso permiten una interacción con el exterior, pues podemos controlar dispositivos que van a tomar acción sobre cosas que pasan a nivel físico y también podemos leer variables del medio ambiente o de otros sistemas que sirven para luego ser analizadas y hecho esto, tomar decisiones para realizar una acción sobre ello. Por esta razón, el Arduino tiene lo que se denominan 6 pines análogos que van desde el A0 hasta el A5, los cuales permiten leer variables continuas del exterior como la luz, la temperatura, entre otros. Estas medidas se hacen a traves de sensores que transforman dichos parámetros en voltaje, el cual es el que ingresa posteriormente al pin y este es el que puede ser leído, se sabe que los pines análogos del Arduino son de 10 bits, por lo que el valor que puede tomar esta entrada está entre 0 y 1023, entonces, una vez se tiene el dato en una variable, se debe de realizar la respectiva conversión para que este valor que ingresó se convierta en una magnitud real. Además, como Arduino no es un sistema embebido tan completo y tan especializado, los pines análogos de este solo se pueden utilizar como entradas más no como salidas. Por último, cabe aclarar que se debe tener cuidado con los valores máximos que se pueden ingresar a estos pines, pues estos son muy sensibles a sobre cargas y si los límites son excedidos se puede quemar todo el Arduino o el conversor análogo-digital.






Sintaxis

Para controlar los pines análogos basta con tener dos cosas en cuenta. Primero se sabe que el pin se puede almacenar en una variable que tenga el nombre del parámetro que se va a leer o se puede poner directamente el número de dicho pin y segundo, para realizar la lectura de cualquiera de estos pines se debe utilizar el comando analogRead(), el cual tiene como parámetro el nombre de la variable donde se almacenó el pin o el número del pin directamente como ya se había mencionado. Una recomendación, es que la lectura de dicho pin se almacene en una nueva variable para posteriormente realizar diferentes operaciones o acciones con ella. 

            
nueva_variable = analogRead(pin_análogo);
