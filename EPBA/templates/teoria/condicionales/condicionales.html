Condicionales


¿Qué son?

Los condicionales son estructuras que permiten comprobar expresiones lógicas, es decir, permiten evalúar si el resultado de dicha expresión es falsa o verdadera y a partir de este resultado, tomar una decisión de si ir por un camino o por otro. De esta manera, esta estructura permite tener un control de flujo de ejecución del programa, lo que hace de esta una herramienta muy potente, pues permite que un programa sea más dinámico y complejo. Dentro de esta estructura se encuentran tres tipos, la condición if , else if y else; cada una de estas cumple una función dentro del control que la estructura brinda. Por ejemplo el if , evalúa directamente si la expresión se cumple y, a partir de esta decisión toma un camino u otro, luego sigue el else if, pues si en el if el resultado fue negativo, la condición else if permite evaluar una nueva expresión y comprobar si esta si se cumple, de igual manera, dependiendo del resutlado tomará un camina u otro. Por último, se tiene el else, el cual brinda la posibilidad de que si no se cumplió ninguno de los anteriores, se ejecutará por ese camino.






Sintaxis

En esta sección se explica un poco de la sintaxis de cada estructura, lo que se va a hacer, será mostrar la estructura y su sintaxis progresivamente para observar cómo al sumar cada tipo de condición el flujo mejora. Para empezar se tiene la sentencia if la cual se muestra a continuación: 

            
if(expresion_logica){
    Instrucciones;
}

Luego, se puede añadir el comando else if, tal y como se muestra en el siguiente fragmento, esto con el fin de poner evidencia un poco de mejor control, el cual brinda la nueva estructura de condición, y así permite elegir el flujo que se desea: 

            
if(expresion_logica){
    Instrucciones;
}
else if(expresion_logica){
    Instrucciones_1;
}

Por último, se agrega la sentencia else, gracias a este se puede tener un control completo del programa puesto que este permite que, dado que no se cumplió ninguna de las expresiones anteriores, el programa tomará este camino y seguirá con la ejecución del programa, es decir, este se puede ver como una opción por defecto. A continuación se muestra cómo: 

            
if(expresion_logica){
    Instrucciones;
}
else if(expresion_logica){
    Instrucciones_1;
}
else{
    Instrucciones_2;
}
