Arreglos


¿Qué son?

Un arreglo o vector, es una estructura de datos o un tipo de dato el cual permite almacenar varios valores a la vez y con la cual se puede realizar, al igual que con cualquier otro tipo de dato, cualquier operación tanto lógica como matemática, obtienendo los mismo valores pero, como ya se mencionó, no solo uno, sino los que se deseen. Otra forma de ver los arreglos unidimensionales es como un vector, es decir que estos pueden interpretar una serie de datos puestas sobre un plano cartesiano, lo que hace que esta sea una herramienta muy potente para realizar programas que se usen por ejemplo, para el análisis de señales o aplicaciones del álgebra lineal.





Sintaxis

Para declarar un arreglo, se hace de manera muy similar a una variable, la diferencia es que a un arreglo hay que especificarle su tamaño y, por otra parte, para darle un valor, se debe hacer entre llaves y el tamaño de este se debe poner entre corchetes. A continuación, se muestra cómo es la estructura y sintaxis correcta de los arreglos: 

            
tipo_de_dato nombre_del_vector[tamaño] = {valor1, valor2, ..., valorN};
